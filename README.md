# RentPredict — ทำนายราคาค่าเช่าหอพักข้างมหาวิทยาลัย

โปรเจคนี้เกิดจากปัญหาที่นักศึกษาหลายคนเจอตอนหาหอพัก คือไม่รู้ว่าราคาที่หอพักตั้งมาเหมาะสมหรือเปล่า เลยลองเก็บข้อมูลหอพักรอบๆ มหาวิทยาลัยมาสร้างโมเดล Multiple Regression เพื่อทำนายราคาค่าเช่าจากปัจจัยต่างๆ เช่น ระยะทาง ขนาดห้อง สภาพห้อง สวัสดิการ ต่างๆ

## ข้อมูลที่ใช้

เก็บข้อมูลหอพักจริง 10 แห่งรอบมหาวิทยาลัย โดยสำรวจด้วยตัวเอง

| Feature | คำอธิบาย | ประเภท |
|---|---|---|
| Distance | ระยะทางจากมหาวิทยาลัย (กม.) | ตัวเลขทศนิยม |
| Room Size | ขนาดห้อง (ตร.ม.) | ตัวเลขทศนิยม |
| Convenience | การเดินทางสะดวกหรือไม่ | 0 = ยาก, 1 = ง่าย/มีวิน |
| Fitness | มีฟิตเนสหรือไม่ | 0 = ไม่มี, 1 = มี |
| Room Condition | สภาพห้อง | คะแนน 1-5 |
| **Price** | **ราคาค่าเช่าต่อเดือน (บาท)** | **Target** |

ข้อมูลตัวอย่างอยู่ในไฟล์ `data/default_data.csv`

## วิธีรัน

```bash
python run.py
```

แค่นี้เลย จะติดตั้ง library ให้อัตโนมัติแล้วเปิดเบราว์เซอร์ไปที่ `http://127.0.0.1:5000`

## วิธีใช้งาน

1. หน้า **Home** → ดูภาพรวมโปรเจคและ features ที่ใช้ในโมเดล
2. เปิดแท็บ **กรอกข้อมูล** → ดูข้อมูลที่มีอยู่ หรือเพิ่ม/ลบข้อมูลได้
3. เปิดแท็บ **Train Model** → กด Train เพื่อสร้างโมเดล จะแสดงค่า R², MAE, MSE, RMSE
4. เปิดแท็บ **ทำนาย** → กรอกข้อมูลหอพักที่สนใจ แล้วกดทำนายราคา

## เทคโนโลยีที่ใช้

- Python, Flask
- scikit-learn (Linear Regression)
- HTML, CSS, JavaScript
